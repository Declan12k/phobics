## About

Phobics is an action-packed shooter game where players face their fears in intense combat scenarios. Built entirely in Python, it combines classic shooter mechanics with modern gameplay elements.

## Features

- Engaging shooter gameplay
- Multiple enemy types
- Power-ups and upgrades
- Score tracking

*More features coming soon!*

## Requirements

- Python 3.8+
- Pygame (or your game library of choice)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/phobics.git
   cd phobics
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the game:
   ```bash
   python main.py
   ```

## How to Play

| Control | Action |
|---------|--------|
| WASD / Arrow Keys | Move |
| Mouse | Aim |
| Left Click | Shoot |
| ESC | Pause |
